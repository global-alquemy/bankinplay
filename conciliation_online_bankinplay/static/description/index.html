Online Conciliation: BankInPlay

Este módulo permite la sincronización bidireccional contable y analítica entre Odoo y Bankinplay.

Configuración

Para el conector:

  1. Ve a Ajustes > Compañías.
  2. Abre la compañía correspondiente.
  3. Cambia a la pestaña Bankinplay.
  4. Establece Api Key y Api Secret.
  5. Guarda la configuración y pulsa Test Connection.

Desde este formulario de configuración podrás exportar manualmente el Plan contable, Plan Analítico (solo de cuentas analíticas seleccionadas), Terceros y Apuntes contables
